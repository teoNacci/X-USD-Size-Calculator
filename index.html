A quick explanation of how this Calculator works. In the main window you will find the three main Inputs and the four main Buttons. Each Input can be filled in by clicking it and typing the desired value. By clicking in the field of an Input you will open an secondary window, below the main one, that contains a second set of inputs related to the main input that has been clicked on. You can use those inputs in order to get the value of the main one. Inside each secondary window you will find two other buttons: the "Submit" button processes the values ​​of the two secondary inputs and sends the result to the main one; the "Reset" button removes any value from the two secondary Inputs. In the main window: the "Calculate" button processes the values ​​of the three main inputs and displays the results in the set of two displays disposed at the bottom; the "Reset" button removes any value ​​from the three main Inputs and the set of two Displays; the "Restore" button recalls the results of the last calculation performed; the "All Clear" button removes any value ​​from three main Inputs, the set of two Displays and all the sets of secondary Inputs. Both displays are also used to visualize errors. Error conditions: The "Calculate" or "Submit" buttons are being clicked with one or multiple Inputs not filled in or with a negative value; superfluous inputs are compiled.
